## Projeto de gerenciamento financeiro pessoal


## Tecnologias

---

- PHP 8.2
- Laravel 11
- Filament 3
- SQLite

---

## Funcionalidades

---

- Cadastro de pessoas
- Cadastro de entradas
- Cadastro de saidas
- Dashboards


---
